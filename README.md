# AETHER Storefront

AETHER is a single-page storefront concept for a Philippine-born technology brand. It showcases products, a cart flow, and a modern blueprint-inspired UI.

## Features
- Single-page navigation (Home, Products, About, Contact)
- Product catalog with detail view
- Cart and checkout flow
- Responsive layout with mobile hamburger menu

## Run Locally
Open `index.html` in your browser.

## Project Structure
- `index.html` - Markup and page sections
- `styles.css` - Visual system and layout
- `script.js` - Interactions, cart logic, and navigation

## Notes
Product images use external URLs for quick prototyping.
